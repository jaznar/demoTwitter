# Code Challenge

* The objective of this challenge is to create a microservice that retrieves tweets from the twitter platform using the twitter4j library for java.
* The persistence will be carried out in a database in memory. H2 (DBMS) .
* Swagger 2

# Configuration

* No configuration needed

# Deployment

* mvn spring-boot:run

# Swagger
* The microservice is documented via swagger after the microservice is started.
* Swagger allows you to make requests through your browser
* http://localhost:8080/swagger-ui.html
